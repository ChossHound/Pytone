Digital Audio Workspaces (DAWs) are powerful tools for music production, but their complexity and cost create significant barriers for beginners. This project presents the design and implementation of a free, beginner-friendly DAW built entirely in Python, with a focus on simplicity, accessibility, and immediate creative feedback. The system uses an object-oriented architecture centered around a piano roll interface, allowing users to visually create and edit notes across multiple tracks. Core components include a Tracklist class that manages tempo, timing, and playback position; Track objects that store notes and assign instruments; Instrument classes that generate distinct sounds for different pitches; and Note objects that represent pitch and duration. A singleton Audio Engine handles sound playback and communicates with the audio device. By simplifying essential DAW features while preserving core music-creation functionality, this project lowers the entry barrier to digital music production and provides an approachable platform for experimentation, learning, and creative expression.
